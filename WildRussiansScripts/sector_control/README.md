# Sector Control — визуальный захват секторов
### Скрипт для Arma 3 | Совместим с локальным и выделенным сервером

---

## Что делает скрипт

- Отображает зоны захвата на карте с линией фронта
- Показывает прогресс захвата и режим боя прямо в названии сектора
- Создаёт задачи для игроков на захват каждого сектора
- Начисляет очки команде при захвате сектора независимыми
- Захват засчитывается только после удержания зоны без боя

---

## Установка

### Шаг 1 — Скопируйте папку скрипта

Скопируйте папку `sector_control` в папку `scripts` вашего сценария:

```
Ваш_сценарий.Карта/
├── init.sqf
└── scripts/
    └── sector_control/
        ├── sc_init.sqf
        ├── sc_sector.sqf
        └── README.md
```

Если папки `scripts` нет — создайте её.

---

### Шаг 2 — Подключите скрипт в init.sqf

Откройте (или создайте) файл `init.sqf` в корне сценария и добавьте строку:

```sqf
execVM "scripts\sector_control\sc_init.sqf";
```

---

### Шаг 3 — Создайте триггеры в редакторе

Для каждого сектора создайте триггер со следующими настройками:

| Параметр      | Значение             |
|---------------|----------------------|
| Тип           | Нет                  |
| Активация     | Нет                  |
| Повторяемый   | ☐ выкл               |
| Только сервер | ☐ выкл               |
| Условие       | `this`               |
| Размер        | Размер вашей зоны    |
| Имя переменной | `trigger_alpha` (уникальное для каждого сектора) |

> Форма триггера (круг или прямоугольник) автоматически переносится на маркер карты.

---

### Шаг 4 — Настройте сектора в sc_init.sqf

Откройте `scripts/sector_control/sc_init.sqf`.  
Это единственный файл, который нужно редактировать.

#### Игровые стороны

```sqf
SC_SIDE_BLUFOR      = WEST;         // Синие
SC_SIDE_OPFOR       = EAST;         // Красные
SC_SIDE_INDEPENDENT = INDEPENDENT;  // Независимые
```

Если в вашем сценарии AAF играют за синих — измените на `INDEPENDENT`.  
Аналогично для любой другой стороны.

#### Задачи по захвату

```sqf
SC_TASKS_ENABLED = true;   // true — задачи создаются, false — нет
```

#### Список секторов

```sqf
private _sectors = [
//   Триггер              Название     Владелец        Очки
    ["trigger_alpha",   "Альфа",    SC_SIDE_OPFOR,  5000],
    ["trigger_bravo",   "Браво",    SC_SIDE_OPFOR,  3000],
    ["trigger_charlie", "Чарли",    SC_SIDE_OPFOR,  7500]
];
```

| Параметр          | Описание                                              |
|-------------------|-------------------------------------------------------|
| `"trigger_alpha"` | Имя триггера из редактора (строка, в кавычках)        |
| `"Альфа"`         | Название зоны на карте                                |
| `SC_SIDE_OPFOR`   | Начальный владелец зоны                               |
| `5000`            | Очки за захват независимыми (0 = без начисления)      |

Доступные значения владельца: `SC_SIDE_BLUFOR`, `SC_SIDE_OPFOR`, `SC_SIDE_INDEPENDENT`, `sideEmpty`

---

## Отображение на карте

| Что видно                    | Описание                                              |
|------------------------------|-------------------------------------------------------|
| Цветные ячейки в зоне        | Линия фронта — кто где стоит                          |
| `Альфа`                      | Зона спокойна, цвет = текущий владелец                |
| `Альфа \| \|\|\|...... 60%`  | Идёт захват, прогресс в процентах                     |
| `Альфа \| БОЙ` (жёлтый)     | В зоне идёт перестрелка, захват заморожен             |
| Серый цвет                   | Зона нейтральная (никем не захвачена)                 |

## Настройки внутри sc_sector.sqf

Эти параметры находятся в начале файла `sc_sector.sqf`.  
Менять их не обязательно, но можно подстроить под свой сценарий.

```sqf
private _gridStep       = 80;  // Шаг сетки фронта в метрах (меньше = точнее)
private _captureTime    = 20;  // Секунд удержания для захвата
private _combatCooldown = 15;  // Секунд тишины до снятия режима БОЙ
private _bgAlpha        = 0.25; // Прозрачность ячеек сетки
```

---

## Требования

- Arma 3
- Для начисления очков: мод **VirtualSquad** (функция `VS_fnc_triggerReward`)  
  Если мода нет — установите `очки_за_захват = 0` для всех секторов

---

## Добавление нового сектора

1. Создайте триггер в редакторе, дайте ему имя (например `trigger_delta`)
2. Добавьте строку в массив `_sectors` в `sc_init.sqf`:
```sqf
["trigger_delta", "Дельта", SC_SIDE_OPFOR, 4000]
```
3. Готово.
