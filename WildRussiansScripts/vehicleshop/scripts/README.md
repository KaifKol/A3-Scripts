# Vehicle Shop — Командный магазин техники для Arma 3

Скрипт добавляет систему командных очков и магазин техники для кооперативных миссий.
Очки зарабатываются за уничтожение противника, тратятся на покупку техники и оборонительных позиций.

ИСПРАВНО РАБОТАЕТ ТОЛЬКО НА ВЫДЕЛЕННОМ СЕРВЕРЕ!!!
замечена проблема взрыва покупаемой техники при установке, если сценарий запущен локально на ПК
---

## Возможности

- Командные очки — общий ресурс отряда
- Магазин техники у доски с картой
- Продажа своей техники с возвратом очков (с учётом здоровья и топлива)
- Сдача трофейной техники за двойную награду
- КШМ — мобильный узел обороны с оборонительными позициями
- Денежные вознаграждения при активации триггеров
- Инструктаж для игроков в дневнике миссии

## Установка

### 1. Скопируй файлы в папку миссии

Структура должна выглядеть так:

```
МояМиссия.Altis/
├── scripts/
│   └── vehicleshop/
│       ├── cfg.sqf
│       ├── server.sqf
│       ├── client.sqf
│       └── ksm.sqf
├── init.sqf
└── description.ext
```

### 2. Добавь в `init.sqf`

Скопируй содержимое `init.sqf` из архива в свой файл.
Если `init.sqf` уже есть — добавь блоки из него в соответствующие места:

```sqf
// В начало файла (серверные обработчики):
if (isServer) then { ... };

// После загрузки cfg.sqf:
execVM "scripts\vehicleshop\cfg.sqf";
waitUntil { !isNil "VS_VEHICLE_LIST" };

// Запуск серверного скрипта:
if (isServer) then {
    [] execVM "scripts\vehicleshop\server.sqf";
    // ... обработчики publicVariable
};

// Запуск клиентских скриптов:
if (hasInterface) then { execVM "scripts\vehicleshop\ksm.sqf"; };
if (hasInterface) then { execVM "scripts\vehicleshop\client.sqf"; };
```

### 3. Добавь в `description.ext`

Скопируй блок `CfgRemoteExec` из файла `description.ext` архива в свой.
Если блок уже есть — добавь только содержимое `class Functions { ... }` внутрь существующего.

```cpp
class CfgRemoteExec {
    class Functions {
        mode = 1; jip = 0;
        class VS_fnc_showHint          { allowedTargets = 0; };
        class VS_fnc_triggerReward     { allowedTargets = 2; };
        // ... остальные классы из description.ext архива
    };
};
```

### 4. Добавь объект доски с картой на базе

В редакторе размести объект класса `Land_MapBoard_F` или `Land_MapBoard_01_F`.
Это точка покупки техники. Радиус действия настраивается через `VS_SPAWN_RADIUS` в `cfg.sqf`.

## Настройка

Все параметры находятся в `scripts/vehicleshop/cfg.sqf`.

### Стартовые очки

```sqf
VS_START_POINTS = 5000;
```

### Список техники

```sqf
VS_VEHICLE_LIST = [
    ["Название в меню", "className", стоимость],
    ["[500] Джип",      "I_MRAP_03_F", 500]
];
```

### Награды за уничтожение

Используются базовые классы Arma 3 (`Plane`, `Helicopter`, `Tank`, `APC`, `Truck`, `Car`, `StaticWeapon`, `Man`):

```sqf
VS_KILL_REWARDS = [
    ["Tank", 2000],
    ["Car",   200]
];
```

### КШМ

Любой транспорт можно назначить КШМ — укажи его className:

```sqf
VS_KSM_CLASS  = "I_MRAP_03_hmg_F";
VS_KSM_RADIUS = 50; // радиус размещения орудий в метрах
```

> **Примечание:** при использовании `I_NFCW_FIN_80_BTR60PU_GM` (Из ДЛС Global Mobilization) при развёртке КШМ будет анимированно подниматься антенна.

### Оборонительные позиции КШМ

```sqf
VS_KSM_DEFENSES = [
    ["Название", "className", стоимость]
];
VS_MAX_DEFENSES_PER_KSM = 10;
```

## Денежные вознаграждения из триггеров

В поле **On Activation** триггера:

```sqf
// Стандартное сообщение:
[500] remoteExec ["VS_fnc_triggerReward", 2];

// Своё сообщение:
[1000, "Точка захвачена!\n+1000 очков команде."] remoteExec ["VS_fnc_triggerReward", 2];
```

---

## Запрет продажи техники как трофея

Чтобы конкретный объект нельзя было сдать как трофей, добавь в его поле **Init**:

```sqf
this setVariable ["VS_noTrophy", true, true];
```

## Совместимость

- Arma 3 версии 2.0 и выше
- Одиночная и мультиплеерная игра (в т.ч. выделенный сервер)
- Не требует модов (опциональная поддержка Northern Fronts для анимации антенны КШМ)
